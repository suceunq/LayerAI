# LayerAI 1.7.28 — Traitement par lot

## Nouveauté

- Un nouveau bouton « Traitement par lot… » dans l'écran d'import permet de mettre en file d'attente plusieurs modèles STL/OBJ/3MF, de les analyser et générer leur configuration à la suite avec l'imprimante, le filament et l'intention actuellement sélectionnés, puis d'exporter tous les profils PrusaSlicer (.ini) en une fois vers un dossier choisi.
